# Superstore Sales Analysis Dashboard

## Overview
This project analyzes Superstore sales data using three BI tools:
- Excel
- Power BI
- Tableau

The goal is to compare dashboard creation across multiple analytics platforms while generating business insights.

## KPIs
- Total Sales: ₹2,297,201
- Total Profit: ₹286,397
- Total Orders: 5009
- Average Discount: 15.62%

## Dashboard Insights
- West region generated highest profit
- Technology category achieved highest sales
- Higher discount reduces profit
- Standard Class contributed highest shipping sales
